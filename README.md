## 🔮 What is it?

**vibe.exe** analyzes a developer's GitHub footprint—languages used, commit times, streak consistency, and repo patterns—and turns it into a highly shareable, visually stunning poster card.

### Key Features
- **Dynamic Organic Blob**: Liquid distortion powered by raw SVG `<feTurbulence>` and `<feDisplacementMap>`.
- **AI-Style Horoscope**: Generates uncanny reads on your coding habits (e.g., "You probably code best after 11pm and regret it every morning").
- **Legendary Stats**: Ranks your commits against the global distribution. Are you in the Top 1% of consistent committers?
- **README Embeds**: Flex your aura on your GitHub profile with a live, auto-updating SVG badge!

## 🚀 Embed in your GitHub README

You can easily embed your aura in your own GitHub profile `README.md`. Just copy and paste this markdown snippet:

```md
<a href="https://vibe-exe.vercel.app">
  <img src="https://vibe-exe.vercel.app/api/aura-badge/YOUR_USERNAME" alt="My Dev Aura" />
</a>
```
*(Replace `YOUR_USERNAME` with your actual GitHub handle!)*

## 💻 Local Development

Want to run it locally or contribute?

```bash
# Clone the repo
git clone https://github.com/0xvikram/vibe.exe.git

# Install dependencies
cd vibe.exe
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to start generating auras.
